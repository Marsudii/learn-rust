# LEARN RUST BASIC
- learn rust basic is a project to learn rust programming language. This project contains basic rust programming language, such as print, variable, type data, and so on.


## list of content
- print
- variable
- type data


### How to run this project
- To run this project, you need to have rust installed on your machine. You can install rust by following the instructions on the official website: [https://www.rust-lang.org/tools/install](https://www.rust-lang.org/tools/install)
- After installing rust, you can run this project by using the following command in your terminal:
```
cargo run
```
